  1. EXPLORATORY DATA ANALYSIS Step 3: First only select the time lagged values and compute Partial Auto-Correlation Function to assert the impact of previous months oil prices on the current price. However as the oil price signal is non-stationary, we take the first-difference to make the time evolution stationary (the variance is however is not IID, can see a clear increase towards the end of the time-frame).
library(ggplot2)

cleaned_price_oil <- forecast::tsclean(df[,1])
first_diff <- diff(cleaned_price_oil)

# Plot the non-seasonal first-order difference of the oil price
plot(first_diff)

It can be observed that we have successfully removed the seasonality.

# Calculate the PACF to check for lag dependancy
pacf(first_diff)

As noticed from the Partial Auto-Correlation function, except for the first lagged value, with 95% probability we can confirm that the influence of old oil price data is minimal on the current price.
